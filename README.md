# Paado Ji Project

A cinematic, editorial-style tribute website for Prachi Jain (also known as "Paado Ji" and "Jiggly Puff").

## Features
- **Cinematic Pacing**: Scroll-driven storytelling using GSAP and ScrollTrigger.
- **Editorial Aesthetic**: Minimalist design with a focus on typography and high-fidelity layouts.
- **Interactive Gallery**: Reveal and parallax animations for a curated dataset of images.
- **Custom Experience**: Custom cursor, grain overlays, and hidden easter eggs.
- **Smooth Interaction**: Integrated with Lenis for premium smooth scrolling.

## Tech Stack
- HTML5 / Vanilla CSS
- Tailwind CSS (CDN)
- GSAP (GreenSock Animation Platform)
- Lenis Smooth Scroll
- Lucide Icons

## Deployment
Prepared for deployment on **Vercel**.
